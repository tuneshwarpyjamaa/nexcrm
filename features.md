# NexCRM — Feature Checklist

> Track all planned features for NexCRM. Checked items (✅) are already present in the project.

---

## 💰 Pricing & Plans

- [ ] Flat, whole-business pricing model (e.g., ₹999–₹4999/mo total, unlimited users) — no per-user scaling trap

## 📦 Core CRM Essentials

- [ ] Unlimited contacts, deals, emails sent, storage, & records in base plan — no caps at 2k–10k or forced tier jumps
- [x] Minimalist, clean, mobile-first UI — no cluttered dashboards/tabs; intuitive like a better Google Sheets
- [x] Email logging & compose — log emails, open in mail client, filter/search email history
- [x] Basic reporting/dashboards — KPI stats, pipeline by stage, contact breakdown, lead sources, task overview
- [x] Task management — create, prioritize, mark done, due dates, task stats
- [x] Notes management — create, edit, delete notes linked to contacts
- [x] Deals pipeline — manage deals across stages (Lead → Won/Lost), deal values, probabilities
- [x] Contact management — full CRUD, tags, status, LinkedIn, company, search & filter
- [x] Activity tracking — recent activity feed on dashboard
- [x] Settings & preferences — business name, user name, email, currency symbol, fiscal year start
- [ ] Email tracking/sync — open tracking, inbox sync, read receipts
- [ ] Email sequences — automated follow-up email chains
- [ ] Task automation — auto-assign, auto-reminders, recurring tasks
- [ ] File attachments — attach files to contacts, deals, notes, emails
- [ ] GST invoicing — generate GST-compliant invoices from deals

## 🛠️ Setup & Onboarding

- [x] Instant self-serve setup (<30 min to productive) — quick register/login flow
- [x] No mandatory onboarding/consultants — self-serve, no forced tutorials
- [ ] Pre-built industry templates (e.g., real estate, agency, shop, freelancer, service biz in India)
- [ ] Free video guides (in English/Hindi)

## 📱 Mobile & Offline

- [x] Full offline mobile app with auto-sync — works in spotty internet areas (key for field teams in Bengaluru/India)

## 🇮🇳 India-Local Integrations

- [ ] WhatsApp Business API (messaging/follow-ups)
- [ ] UPI/Razorpay/Paytm payment links in deals/invoices

## 📊 Data Management

- [x] Easy full data export anytime (JSON) — no vendor lock-in fear
- [x] Import backup from JSON file
- [x] Danger zone — clear contacts/deals/tasks/everything, delete account
- [ ] One-click import + auto-cleanup from Excel/Google Sheets/old tools — no data headaches or paid migration services

## ⚡ Automation & Intelligence

- [ ] Drag-and-drop simple automations — no code/Flow Builder required; e.g., auto-assign by pincode/area, reminders, birthday/SMS
- [ ] Basic practical AI (optional wow factor) — e.g., suggested next actions, overdue alerts — without complex/expensive Einstein-style add-ons

## 🌐 Localization & Accessibility

- [ ] Regional language support (Hindi + others)
- [ ] Voice note/transcription for calls/notes

## 💬 Support

- [ ] Fast, human support channels — WhatsApp/chat priority (no slow tickets or 500-page docs)

## 📅 Collaboration & Scheduling

- [ ] Shared team calendar + auto-meeting scheduler built-in

## 🏪 Customer Portal

- [ ] Customer self-service portal basics (view deals/invoices, pay) — rare in cheap tools

## 🔓 No Hidden Limits

- [ ] No hidden limits on core actions — unlimited records, no API throttling for basics
- [ ] Unlimited custom fields/reports in base plan
